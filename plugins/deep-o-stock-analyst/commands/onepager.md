# /deep-o-stock-analyst:onepager

**One-Pager** — สรุปทุกแง่มุมเป็นเรื่องเล่าภาษาง่าย มือใหม่ฟังรู้เรื่อง

## Input ที่ต้องการ

- ผลวิเคราะห์ DEEP+O ที่ทำไว้ (จาก `/full` หรือ command อื่น) — ถ้ายังไม่มี ให้ขอ ticker แล้วบอกว่าควรรัน `/full` ก่อน

## สิ่งที่ทำ

- เล่าเป็น **เรื่องเดียวต่อเนื่อง ไม่มี bullet** — เหมือนนั่งฟังหุ้นส่วนรายงานในห้องประชุม
- ครอบทุกแง่มุม: ธุรกิจทำอะไร · ดีมานด์โตจากไหน · ทำเงินคุ้มต้นทุนทุนไหม · ราคาวันนี้แพง/ถูกเทียบมูลค่า · ความเสี่ยงใหญ่สุด · สรุปคำตัดสิน
- **ย้ำว่าภาษาต้องง่าย** — เลี่ยงศัพท์ ถ้าใช้ต้องขยายความ ให้คนไม่รู้เรื่องหุ้นก็ตามได้

## ตัวอย่างสั่ง

```
/onepager
```

## Discipline

เล่าตามข้อเท็จจริงที่วิเคราะห์มา ไม่เพิ่มตัวเลขใหม่ที่ไม่มีแหล่ง · ระบุ **as-of date** · verdict = framework signal เชิงการศึกษา ไม่ใช่คำแนะนำรายบุคคล
